# data
- `type`: `Array`
- `default`: `[]`

图表的数据。

目前只支持一种数据结构：

```javascript
const data = [
  { genre: 'Sports', sold: 275, age: 10, team: 'team1' },
  { genre: 'Strategy', sold: 115, age: 20, team: 'team1' },
  { genre: 'Action', sold: 120, age: 20, team: 'team2' },
  { genre: 'Shooter', sold: 350, age: 10, team: 'team2' },
  { genre: 'Other', sold: 150, , age: 10, team: 'team3' },
];
```
